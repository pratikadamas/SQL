# Question
Show all the columns from admissions where the patient was admitted and discharged on the same day.
# Query

```sql
SELECT 
    p.patient_id,
    a.admission_date,
    a.discharge_date,
    a.diagnosis,
    a.attending_doctor_id
FROM patients p
JOIN admissions a 
    ON p.patient_id = a.patient_id
WHERE a.admission_date = a.discharge_date;
```
