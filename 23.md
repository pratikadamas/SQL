# Question

Display the total amount of patients for each province. Order by descending.

# Query

```sql
SELECT 
	p1.province_name,count(p2.province_id) as patient_count
    from patients p2
    join province_names p1
    on p2.province_id=p1.province_id
    group by p2.province_id
    order by patient_count desc;
```
